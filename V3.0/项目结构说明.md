# TimeMaster V3.0 项目结构

> 项目名: TimeMaster (时间管理大师)
> 命名空间: `timemaster`
> Qt 6 / C++17 / CMake

## 目录结构

```
V3.0/
├── CMakeLists.txt          # 构建配置（TimeMaster 2.0.0）
├── README.md               # 项目说明
├── build_windows.bat       # Windows 一键构建脚本
└── src/
    ├── main.cpp            # 程序入口
    ├── core/               # 核心层
    │   ├── Types.h/.cpp    # 数据类型定义（CalendarEvent, AiBatchInfo...）
    │   ├── Database.h/.cpp # SQLite 数据库（含 AI 批次管理/撤销支持）
    │   └── DeepSeekClient.h/.cpp  # DeepSeek AI API 客户端
    └── ui/                 # UI 界面层
        ├── Theme.h/.cpp    # 主题系统（高级玻璃质感）
        ├── MainWindow.h/.cpp      # 主窗口（带渐变背景层）
        ├── Sidebar.h/.cpp         # 侧边导航栏
        ├── CalendarPage.h/.cpp    # 日历主页（含 AI 批次跟踪）
        ├── ChatPage.h/.cpp        # AI 对话页（含日历上下文注入）
        ├── AnalyticsPage.h/.cpp   # 数据分析页
        ├── MonthView.h/.cpp       # 月视图
        ├── TimeGridView.h/.cpp    # 时间网格视图
        ├── EventDialog.h/.cpp     # 事件编辑对话框
        ├── SettingsDialog.h/.cpp  # 设置对话框
        ├── AiHistoryDialog.h/.cpp # AI导入历史（撤销/删除）
        ├── CategoryPieChart.h/.cpp # 分类饼图
        └── widgets/                # 分析页子组件
            ├── StatsCardsWidget.h/.cpp
            ├── HorizontalBarChart.h/.cpp
            ├── DailyTrendChart.h/.cpp
            ├── RhythmCardWidget.h/.cpp
            ├── SourceDistributionWidget.h/.cpp
            └── InsightsWidget.h/.cpp
```

## 文件完整性

CMakeLists.txt 列出的 43 个源文件全部完整存在，包括：
- `src/core/` — 6 个文件 (3 .h + 3 .cpp)
- `src/ui/` — 24 个文件 (12 .h + 12 .cpp)
- `src/ui/widgets/` — 12 个文件 (6 .h + 6 .cpp)
- `src/main.cpp` — 1 个文件

## V3.0 相对于 V2.x 的改进

1. **项目更名**: TimePlan → TimeMaster, 命名空间 `timeplan` → `timemaster`
2. **UI 升级**: "高级玻璃质感"主题（rgba 半透明 + 径向渐变光晕）
3. **AI 导入历史**: AiHistoryDialog 面板，支持单条/整批撤销 AI 导入的日程
4. **日历上下文注入**: ChatPage 在发送前自动将日历事件注入 AI 上下文
5. **数据库升级**: 新增 ai_batches 表追踪 AI 导入批次

## V2.0 存档

旧版 V2.x 的完整源码（命名空间 `timeplan`）已归档到 `V2.0/src/` 目录下。

## 构建方式

```bash
cd V3.0
mkdir build && cd build
cmake ..
cmake --build .
```

Windows 可按需运行 `build_windows.bat`（需已安装 Qt6 + MinGW）。
